
# 🎟️ Smart Ticket Booking System

A full-stack web application for booking event tickets with real-time seat selection, user-friendly UI, and efficient backend handling.

---

## 📌 Features

- 🎫 Browse available events
- 🪑 Real-time seat selection
- 🔒 Seat locking system to prevent double booking
- ⏳ Timer for seat reservation
- 📊 Admin dashboard (basic)
- 📱 Responsive UI design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Other Tools
- Git & GitHub
- REST APIs
